
## ENV
* APP_NAME：应用名称：web、service、consumer，etc...
* LOGGER_CONF_PATH：日志配置文件位置
* CACHE_CONF_PATH：缓存配置文件位置
* MYSQL_CONF_PATH：数据库配置文件位置

* WEB_HOST：WEB应用的监听HOST
* WEB_PORT：WEB应用的监听PORT
